🔋 ESP32 Smart Shunt для LiFePO4 (24V 200Ah)
Інтелектуальна система моніторингу акумуляторів на базі ESP32-C3 та модуля INA226. Проєкт дозволяє перетворити звичайний шунт на "розумний" пристрій, що інтегрується в Home Assistant через ESPHome.

Ідеально підходить для гібридних інверторів (наприклад, PowMr) та самозбірних LiFePO4 акумуляторів.

🚀 Основні можливості
Точне вимірювання: Напруга (V), Струм (A) та Потужність (W) у реальному часі.

Розрахунок ємності (SoC): Відстеження залишку заряду у % та Ампер-годинах (Ah).

Інтелектуальний таймер: Розраховує час до повного розряду або повного заряду залежно від поточного навантаження.

Захист від накопичення похибки: Автоматичне скидання лічильника на 100% при завершенні циклу зарядки від мережі.

Цифрова стабілізація: Апаратне та програмне усереднення (sliding window moving average) для плавних графіків без стрибків.

Адаптивний статус: Динамічна зміна назви стану ("До розряду" / "До повного заряду").

🛠 Компоненти
Мікроконтролер: ESP32-C3 (або будь-який ESP32).

Сенсор: INA226.

Шунт: Зовнішній 100A / 75mV (опір у коді 0.00075 ohm).

АКБ: LiFePO4 збірка 24В 200Аг.

Платформа: Home Assistant + ESPHome.

📦 Встановлення та налаштування
1. Прошивка ESPHome
Скопіюйте вміст файлу smart_shunt.yaml до вашого проєкту ESPHome.

Порада: Перед використанням перевірте напругу на клемах мультиметром та внесіть корективи у блок calibrate_linear, щоб досягти максимальної точності.

2. Налаштування Dashboard
Для створення візуального інтерфейсу використовуйте плагін custom:button-card. Код інтерфейсу знаходиться у файлі dashboard.yaml.

⚙️ Схема підключення (Спрощена)
I2C шина: SDA (пін 8), SCL (пін 9).

Шунт: Встановлюється в розрив мінусового дроту між акумулятором та інвертором.

V-BUS: Дріт від піна V-BUS модуля INA226 підключається до плюсової клеми АКБ (обов'язково через запобіжник 1А).

📈 Калібрування
Якщо ви бачите розбіжність між показниками інвертора та ESPHome, використовуйте фільтр калібрування в коді:

YAML
calibrate_linear:
  - 12.0 -> 17.0  # (Значення, яке бачить ESPHome -> Реальне значення)
